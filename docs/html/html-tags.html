HTML Tags​

Tags plays an important role in web development, tags hold content to display their are specific tags for specific texts, like if you want to add paragraph then use <p> </p> tags for heading <h1> </h1> and many more...

Closing Tags​

Closing tags are those tags which have opening and closing tags both, like as we discuss above that if you have to add any paragraph you will add a <p> tags as opening and </p> as closing tag,

<p>
    This is my paragraph
</p>

Opening Tags​

Opening tags are those tags which have only opening tags no closing tag, actually they also have closing tag bt we don't need to type it like if we want to add a line break in our webpage we simply add <br> but we will not add </br> because this closing tags don't have any use, these kinds of tags are self closing tags as we can also just type it as <br/> which indicate both opening and closing tag as well.

This is my text and now i have to break a line and continue to next line.
<br> <!-- or <br/> -->
Now this is my new line
